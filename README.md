# FreeCodeCamp - Data Analysis with Python Projects

This repository contains my solutions for the FreeCodeCamp Data Analysis with Python Certification. 

## Projects Completed

### 1. Mean-Variance-Standard Deviation Calculator
- **File**: `mean_var_std.py`
- **Description**: Uses NumPy to calculate mean, variance, standard deviation, max, min, and sum for a 3x3 matrix from a list of 9 digits.
- **Skills**: NumPy, matrix operations

### 2. Demographic Data Analyzer  
- **File**: `demographic_data_analyzer.py`
- **Dataset**: `adult.data.csv`
- **Description**: Analyzes demographic data using Pandas. Answers questions about race, age, education, and salary.
- **Skills**: Pandas, data cleaning, filtering

### 3. Medical Data Visualizer
- **Files**: `medical_data_visualizer.py`, `medical_examination.csv`
- **Output**: `catplot.png`, `heatmap.png`
- **Description**: Visualizes medical examination data using matplotlib and seaborn. Creates categorical plots and correlation heatmaps.
- **Skills**: Matplotlib, Seaborn, data visualization

### 4. Page View Time Series Visualizer
- **Files**: `time_series_visualizer.py`, `fcc-forum-pageviews.csv` 
- **Output**: `line_plot.png`, `bar_plot.png`, `box_plot.png`
- **Description**: Visualizes forum page views using line charts, bar charts, and box plots to show trends over time.
- **Skills**: Pandas, Matplotlib, time series analysis

### 5. Sea Level Predictor
- **Files**: `sea_level_predictor.py`, `epa-sea-level.csv`, `epaa-sea-level.csv`
- **Output**: `sea_level_plot.png`
- **Description**: Predicts sea level change through 2050 using linear regression on historical data.
- **Skills**: SciPy, linear regression, forecasting

## Tech Stack
- **Language**: Python 3
- **Libraries**: NumPy, Pandas, Matplotlib, Seaborn, SciPy
- **IDE**: VS Code

## How to Run
1. Clone this repo: `git clone https://github.com/Suvedha-K/freecodecamp-mean-var.git`
2. Install requirements: `pip install numpy pandas matplotlib seaborn scipy`
3. Run any project: `python mean_var_std.py`
